# BayerDBA 项目规划

**项目名称:** BayerDBA - 交互式 DBA 技能考察系统  
**创建时间:** 2026-03-08  
**目标:** 针对 DBA 岗位需求，提供交互式技能测评和针对性练习

---

## 📋 项目背景

根据 Bayer DBA 岗位需求，设计交互式技能考察系统，帮助候选人/在职 DBA 评估和提升以下核心技能：

### 核心技能领域

| 技能领域 | 权重 | 考察重点 |
|----------|------|----------|
| **AWS 云技能** | 35% | IAM、权限、安全、计算 (EC2/Lambda)、存储 (S3/EBS)、网络 (VPC) |
| **Python 技能** | 30% | API 调用、脚本开发、运维自动化 |
| **数据库技能** | 35% | 权限管理、安全管理、备份管理、性能优化 |

---

## 🎯 功能设计

### 1. 技能测评模式

**流程:**
```
选择技能领域 → 等级测评 (10 题) → 评估等级 → 生成学习建议
```

**等级划分:**
| 等级 | 分数范围 | 描述 |
|------|----------|------|
| L1 - 初级 | 0-40 分 | 基础概念了解，需要系统学习 |
| L2 - 中级 | 41-70 分 | 掌握核心技能，可独立完成常规任务 |
| L3 - 高级 | 71-85 分 | 深入理解，能解决复杂问题 |
| L4 - 专家 | 86-100 分 | 精通领域，可指导他人 |

### 2. 专项练习模式

**流程:**
```
选择技能点 → 难度选择 → 交互式练习 → 即时反馈 → 错题记录
```

**功能:**
- ✅ 选择题/填空题/实操题
- ✅ 即时正误判断
- ✅ 知识点解析
- ✅ 错题自动记录

### 3. 错题集管理

**功能:**
- 📝 自动记录错题
- 🔄 间隔重复提醒 (1 天、3 天、7 天)
- ✅ 掌握标记 (连续 2 次正确)
- 📊 掌握率统计

**错题状态:**
```
新错题 → 待复习 (1 天后) → 复习中 → 已掌握 (连续 2 次正确) → 归档
```

### 4. 进度跟踪

**跟踪内容:**
- 各技能领域等级变化
- 练习题目数量统计
- 错题掌握率趋势
- 学习时长记录

## 🎓 题库设计

### AWS 云技能题库

**IAM & 权限 (15 题)**
- IAM 用户/组/角色概念
- 策略文档结构
- 最小权限原则
- 跨账号访问
- 临时凭证

**安全 (15 题)**
- KMS 加密
- Security Groups vs NACL
- WAF 配置
- CloudTrail 审计
- GuardDuty 威胁检测

**计算 (10 题)**
- EC2 实例类型
- Auto Scaling
- Lambda 函数
- ECS/EKS 容器

**存储 (10 题)**
- S3 存储类
- EBS 卷类型
- EFS 文件系统
- 数据生命周期

**网络 (10 题)**
- VPC 子网划分
- 路由表配置
- NAT Gateway
- VPC Peering
- Direct Connect

### Python 技能题库

**API 调用 (15 题)**
- requests 库使用
- REST API 认证
- 错误处理
- 速率限制处理
- 异步调用

**脚本开发 (15 题)**
- 文件操作
- 正则表达式
- 数据处理 (JSON/YAML)
- 日志记录
- 参数解析

**运维自动化 (10 题)**
- boto3 AWS SDK
- subprocess 调用
- 定时任务
- 监控告警
- 配置管理

### 数据库技能题库

**权限管理 (15 题)**
- 用户/角色创建
- GRANT/REVOKE
- 细粒度权限
- 审计日志

**安全管理 (15 题)**
- 加密 (TDE/SSL)
- 密码策略
- 访问控制
- 漏洞管理

**备份管理 (15 题)**
- 备份策略
- 全量/增量备份
-  PITR 恢复
- 备份验证
- 灾难恢复

**性能优化 (10 题)**
- 索引优化
- 查询计划分析
- 锁机制
- 参数调优
